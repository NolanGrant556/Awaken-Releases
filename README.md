# Awaken Releases

Official public downloads for Awaken Desktop.

Each published release may contain:

- Awaken Desktop installers for supported platforms and architectures
- Awaken managed Work Runtime archives
- Runtime catalog and integrity manifests
- Release notes and compatibility information

The Awaken source repository remains private. This repository contains release artifacts only.

No production release is currently available. Development or ad-hoc builds must not be treated as signed production releases.
